# React port of Backbone's [local-todos](http://backbonejs.org/docs/todos.html) sample app

## Demo

https://yuliskov.github.io/react-localtodos/

## Features

- Webpack support
- Styl support
- Redux
- Undo your steps

## Start the demo server

```console
$ git clone git@github.com:yuliskov/react-localtodos.git
$ cd react-localtodos
$ npm install
$ npm start
```
